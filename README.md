# Preact Basic Website

A minimal Preact + Vite starter.

Quick start (zsh / macOS):

```bash
cd /Users/karangill/Documents/CloseTechAi
npm install
npm run dev
```

- Open: `http://localhost:5173`
- Build: `npm run build`
- Preview build: `npm run preview`

If you prefer `pnpm` or `yarn`, substitute `npm install` accordingly.
